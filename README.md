## Quick Start

1. Pick a template that matches your target client
2. Copy the workflow to your automation platform (n8n/Make/Zapier)
3. Update the prompts with your specifics
4. Test with sample data
5. Go live and scale

## Prerequisites

- n8n or Make.com account (free tier works)
- OpenAI API key (Claude also works)
- Target platform connections (LinkedIn, email tools, etc.)
